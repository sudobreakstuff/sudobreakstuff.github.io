Profile

I build systems that make other systems work better. At Jenny Internet, I design and develop internal software — ticket automation workflows, Chrome extensions for support agents, file management tools — that keep support teams running efficiently.

I build things because of problems. Screenshots are a pain? I build an extension. A bakery can't afford a POS? I build SpiceDesk and open-source it. Macro keyboards cost too much? I design my own and sell it for R1,099.

Every project starts with "this shouldn't be this hard" and ends with code anyone can use. Open source isn't a license — it's a philosophy. Why paywall a tool that solves a real problem?

If it's repetitive, I automate it. If it doesn't exist yet, I build it.
